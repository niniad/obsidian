# 絵本アイデアを考えるプロンプト（ChatGPT4o）

私は商品開発担当です。開発する育児関連商品のデザインを絵本の一幕にするというコンセプトを持っています。まずは、ストーリーとキャラクターを作成し、それを表す商品を開発したいと思います。以下の傾向と具体的な要件を参考に幼児向け絵本のストーリーを考えてください。

# 傾向と要件

## ストーリーとキャラクター

売れる絵本のストーリーやキャラクターには、以下のような特徴が見られます:

- **シンプルで分かりやすいプロット**: 幼い読者に理解しやすい単純な物語構造が好まれます。
- **共感できる主人公**: 子どもが自分を投影しやすいキャラクターが人気を集めます。
- **繰り返しのパターン**: リズミカルな繰り返しや予測可能な展開が子どもの興味を引きます。
- **教育的要素**: 道徳的メッセージや学習要素を含む物語が評価されます。

## 視覚的要素

絵本の視覚的な側面も重要な役割を果たします:

- **鮮やかな色使い**: 明るく鮮やかな色彩が子どもの注目を集めます。
- **大きくはっきりとしたイラスト**: 細部まで描き込まれた絵よりも、シンプルで大きな絵が好まれる傾向があります。
- **一貫したスタイル**: 絵のスタイルが本全体で統一されていることが重要です。

## その他の傾向

売れる絵本には、以下のような特徴も見られます:

- **適切な長さ**: 子どもの集中力に合わせた適度な長さが重要です。
- **読み聞かせやすさ**: 大人が読み聞かせやすい文章や構成が評価されます。
- **インタラクティブ要素**: めくり絵や仕掛け絵本など、子どもが参加できる要素が人気です。

# 具体的な指示

1. **ストーリーのアイデア**: 主人公が家の中や庭で冒険しながら、何かを探す物語を考えてください。各シーンで新しいキャラクターに出会い、一緒に探す過程を描いてください。
2. **キャラクターの設定**: 主人公は、やんちゃで元気いっぱいの小さな動物や妖精など。友達として、知恵のあるキャラクターやユーモラスなキャラクターを追加してください。
3. **視覚的要素**: 各シーンは鮮やかな色彩で描かれ、キャラクターは大きくはっきりと描いてください。絵のスタイルは一貫してシンプルで、キャラクターの表情や動きを強調してください。
4. **商品デザインへの反映**: 絵本の各シーンを基にしたウェットティッシュケースのデザインを提案してください。例えば、キャラクターのイラストや物語の一幕をデザインに取り入れてください。
