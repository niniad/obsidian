## Order Reports
Order reports provide order information for the Selling Partner.

### GET_FLAT_FILE_ALL_ORDERS_DATA_BY_ORDER_DATE_GENERAL
Contains all orders placed in the specified date range.
- **amazon-order-id**
- **merchant-order-id**
- **purchase-date**
- **last-updated-date**
- **order-status**
- **fulfillment-channel**
- **sales-channel**
- **order-channel**
- **url**
- **ship-service-level**
- **product-name**
- **sku**
- **asin**
- **item-status**
