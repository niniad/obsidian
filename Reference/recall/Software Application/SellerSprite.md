セラースプライト (SellerSprite) は、Amazon 出品者向けのリサーチ・分析ツールです。キーワードリサーチ、キーワードトレンド、ライバル商品リサーチ、市場リサーチ、商品リサーチ、キーワードマイニング、キーワード逆引き、キーワード順位チェッカー、キーワードモニタリング、商品モニタリング、販売数予測など、Amazon ビジネスに必要な情報をブラウザ上で取得できます。スマートフォンのブラウザでも利用可能で、Google Chrome 用の拡張機能（Chrome Web Store）も提供されています。無料プランと有料プラン（スタンダード、アドバンスド、VIP）があり、月額または年額で利用できます。
