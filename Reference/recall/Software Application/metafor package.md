The metafor package is a comprehensive collection of functions for conducting meta-analyses in R. It provides tools to calculate various effect sizes or outcome measures, fit equal-, fixed-, random-, and mixed-effects models, perform moderator and meta-regression analyses, and generate meta-analytical plots such as forest, funnel, and bubble plots.
