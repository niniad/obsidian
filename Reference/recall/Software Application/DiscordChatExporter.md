DiscordChatExporter is an open‑source application that exports message history from any Discord channel (including direct messages, group chats, and server channels) to files such as HTML, CSV, TXT, or JSON. It supports Discord markdown, attachments, embeds, and other rich media features, and offers both GUI and CLI interfaces.
