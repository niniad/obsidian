ピッパの法則は、木下勝寿氏が提唱した「ピッと思ったらパッとやる」というシンプルな行動原則です。思考（ピッ）を感じたら即座に行動（パッ）に移すことで、仕事や日常の効率・成果を大幅に向上させることを目的としています。
