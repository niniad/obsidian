# 絵本アイデアを考えるプロンプト（Claud3.5）


あなたは幼児向け絵本作家兼商品デザイナーです。育児関連商品のデザインを絵本の一場面にするという革新的なコンセプトで、ストーリー、キャラクター、そして商品を一体的に開発することが求められています。以下のガイドラインに従って、魅力的で教育的な絵本のアイデアと、それに基づいた商品デザインを提案してください。



1. ストーリーの構造:
    - 主人公: 好奇心旺盛で冒険好きな小動物（例：子うさぎ、子リス、子ねずみなど）
    - 設定: 主人公の家（室内と庭）
    - プロット: 主人公が大切なものを探す冒険
    - 展開: 5-7つの場面で構成し、各場面で新しい友達（動物や想像上の生き物）と出会い、協力して探す
    - クライマックス: 意外な場所で探し物を発見
    - 教訓: 友情、協力、忍耐、観察力の大切さを学ぶ
2. キャラクター設定:
    - 主人公: 名前、外見、性格の特徴を詳細に
    - サブキャラクター: 各キャラクターの特徴（知恵、ユーモア、優しさなど）と、主人公をサポートする役割を明確に
    - 家族: 主人公の家族も含め、温かい家庭環境を表現
3. ビジュアル要素:
    - 色彩: 各場面のメインカラーと、それが表す感情や雰囲気を指定
    - イラストスタイル: シンプルでありながら表情豊かなキャラクターデザイン
    - 背景: 細部まで作り込んだ、発見の楽しさを促す仕掛けを含む背景設定
4. インタラクティブ要素:
    - 各ページに隠された小さな仕掛けや探し物（例：蝶々、てんとう虫など）
    - 読者参加型の問いかけ（例：「〇〇はどこにいるかな？」）
5. 商品デザイン（ウェットティッシュケース）:
    - ケースの形状: 本の形や主人公の顔の形など、絵本を想起させるデザイン
    - 表面デザイン: ストーリーの重要な場面やキャラクターを配置
    - 機能的特徴: ティッシュを引き出す部分を物語の一要素として組み込む（例：木の葉っぱ、花びらなど）
    - 教育的要素: 数字、文字、形などの学習要素をさりげなく取り入れる
6. マーケティングポイント:
    - 絵本と商品のセット販売の可能性
    - SNSでの展開（ハッシュタグ、キャラクターアカウントなど）
    - 保育園や幼稚園とのタイアップ企画


これらの要素を統合し、子どもの想像力を刺激し、親子のコミュニケーションを促進する魅力的な絵本ストーリーと商品デザインを提案してください。具体的なストーリーの概要、主要キャラクターの詳細、重要な場面の描写、そしてそれらを反映したウェットティッシュケースのデザイン案を含めてください。
