# 遺伝が9割である理由と根拠は？


- 複数の研究から遺伝子が子供の行動に大きな役割を果たすこと。遺伝子の影響は育児方法の影響よりも大きいこと示されている。
- 親の育て方と子供の行動の相関関係がある事は判明しているが、因果関係は明らかになっていない。子供の行動が親の育て方に影響している可能性もあり、親子が共有する遺伝子によって相関する可能性もある。実際にはほとんどの親子研究では、子育ての効果についてはほとんどわかっていない。
- 養子研究では、遺伝子の影響と家庭環境の影響を切り離すことができる親子の類似性を調べた研究では、子供は圧倒的に実の親に似ていることがわかっており、行動に遺伝的な影響があることを示す強力な証拠となっている。
- 双子研究では一卵性双生児と二卵性双生児を比較することで、遺伝的な影響と環境的な影響を比較している。その結果一卵性双生児は、二卵性双生児よりもあらゆる行動において似ていることがわかっている。また、異なる環境で育った一卵性双生児の相関性は、同じ親に育てられた一卵性双生児の相関性とほぼ同じであることもわかっている。





# 子供の気質の見極め方


- 外交性が高い子供
    - スリルあるゲームがすき
    - 新しい場所を探索するのが好き
    - 知らない人に会うのが好き
    - エネルギーに満ち溢れている
    - 読書のような静かな活動より元気いっぱいに駆け回る活動を好む
    - 初対面の人に打ち解けるまでに時間がかからず、新しい状況に慣れるのが早い
- 情動性が高い子供
    - 物事が思うように進まないととてもイライラする
    - お化けや夜中になると怖がる
    - 一旦怒ると、10分以上たっても、怒りが収まらない
    - 動揺したり、怒ったりしたときになだめたり、気持ちを切り替えさせたりするのが難しい
    - 計画通りに進まなかったときに動揺する
    - 動揺すると、立ち直りが遅く、別の活動に方向転換できない
- 自制心(自己制御能力）の高い子供
    - ダメと言われれば、その行動を止めることができる
    - 塗り絵や紬など、1つの活動に没頭しているときには、とても高い集中力を発揮する
    - 指示に従うことができる
    - 危険だと言われた状況に近づくと、注意深くなる
    - 待てと言われれば、ご褒美を待つことができる
    - 順番をまったり、じっとしていることができる
    - よく考えて行動し、状況に飛び込んでしまうことがない
